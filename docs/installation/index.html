Installation & Onboarding - node-pptx-templater

NPM Package Installation

Install the library directly into your project using npm:

npm install node-pptx-templater

Prerequisites

Ensure your environment meets the minimum requirements:

  • Node.js version 18.x or higher
  • npm version 8.x or higher
  • Compatible with TypeScript, CommonJS, and ES modules
  • Zero native binaries or Microsoft Office dependencies
