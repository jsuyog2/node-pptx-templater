Label Templates - node-pptx-templater

Dynamic Label Templates

Templates allow you to mix variables and custom text to build descriptive labels. The templating engine evaluates expression templates for each data point and writes them as text values in both the chart XML and the embedded workbook.

Supported Template Variables

  • {category}: Evaluates to the data point's category name (e.g. "Sales").
  • {value}: Evaluates to the numeric value of the point (e.g. 150).
  • {percentage}: Evaluates to the percentage of the point relative to the series sum (useful for pie/bar charts).
  • {series}: Evaluates to the series name.
  • {customLabel}: Evaluates to the literal label or category mapped label.

Usage Example

ppt.useSlide(1).updateDataLabels('KPIChart', {
  series: 0,
  template: '{category}: {value} ({percentage}%)'
});
