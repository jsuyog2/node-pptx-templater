Images API Reference - node-pptx-templater

Add images dynamically, swap templates, and extract file lists.

Included API Methods

  • replaceImage — Replaces an existing image in the presentation by shape name or relationship ID.
  • addImage — Adds a new image to the targeted slide(s) at the specified position.
  • removeImage — Removes an image from the targeted slide(s) by shape name or relationship ID.
  • getImages — Returns metadata for all images found on the targeted slide(s).
