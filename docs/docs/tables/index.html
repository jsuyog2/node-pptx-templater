Tables API Reference - node-pptx-templater

Manipulate DrawingML tables, rows, cells, auto-fitting, resizing, and cell merging.

Included API Methods

  • updateTable — Replaces table rows with new data in the selected slide(s). Preserves borders, merged cells, fonts, colors, and alignment from the template.
  • getTableRows — Extracts table data from the active slide as structured JSON. The first row of the table is treated as the header row.
  • addTableRow — Appends one or more rows to a table. Supports flat arrays and nested arrays for rowspan-merged cells. `'rowspan'` creates OpenXML vertical spans, `'auto'` merges identical adjacent values, `'none'` expands nested arrays into multiple flat rows.
  • removeTableRow — Removes a row from a table by its 0-based row index.
  • insertTableRow — Inserts a new row at the specified 0-based index, shifting existing rows down.
  • cloneTableRow — Clones a row and inserts the copy at a target position.
  • updateCell — Updates the text and optional formatting of a single table cell.
  • mergeCells — Merges a rectangular region of table cells into a single merged cell. Supports both positional arguments and an options object.
  • unmergeCells — Unmerges (splits) a previously merged cell region. Supports both positional arguments and an options object.
  • getMergedCells — Returns an array of all merged cell regions in a table.
  • validateMergeRegion — Validates whether a merge region is valid for the given table dimensions. Checks for overlapping merges, out-of-bounds coordinates, etc.
  • isMergedCell — Checks whether a specific table cell is part of a merged region.
  • getMergeParent — Returns the anchor (parent) cell coordinates of a merged region that contains the given cell.
  • getMergeRegion — Returns the full extent of the merged region containing a given cell.
  • splitMergedRegion — Splits a previously merged cell region back into individual cells.
  • cloneMergedRegion — Clones an existing merged region to a new anchor position in the table.
  • autoFitTable — Automatically adjusts column widths to fit the content of each cell.
  • resizeTable — Resizes a table to the specified width and height in EMUs. 1 inch = 914,400 EMUs.
  • getTables — Returns metadata for all tables on the targeted slide(s).
  • addCellShape — Dynamically adds a shape inside a table cell based on cell coordinates. Cell shapes are overlay graphics anchored independently of the table layout, and adding a cell shape never modifies row heights, column widths, or table dimensions.
  • updateCellShape — Updates an existing shape inside a table cell.
  • removeCellShape — Removes a shape from a table cell.
  • getCellShape — Discovers and retrieves details of an existing cell shape on the targeted slide.
  • getCellBounds — Retrieves final rendered bounds of a table cell in pixels.
  • getCellPosition — Retrieves final rendered position of a table cell in pixels. Optionally calculates centered top-left coordinates for a shape of given dimensions.
