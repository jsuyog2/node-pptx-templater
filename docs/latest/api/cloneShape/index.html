cloneShape() API Method - node-pptx-templater

Description: Duplicates an existing shape with a new ID, optionally at a different position.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
shapeId string Source shape name or ID.
newShapeId string Name/ID for the cloned shape.
[options] Object Position overrides for the clone.
[options.x] number X offset for the clone (EMUs).
[options.y] number Y offset for the clone (EMUs).

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Duplicates the target shape element XML node (`p:sp`) and adds positioning coordinates.

Edge Cases & Error Handling

Edge Cases: Offsets are in English Metric Units (EMU). Offset parameters specify position displacement.

Errors: Throws error if target shape is not found on active slide.

Code Example

ppt.cloneShape('logo', 'logo-copy', { x: 2000000, y: 500000 });
