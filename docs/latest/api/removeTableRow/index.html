removeTableRow() API Method - node-pptx-templater

Description: Removes a row from a table by its 0-based row index.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
rowIndex number 0-based row index to remove.

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Removes the target `<a:tr>` node entirely from the slide table XML.

Edge Cases & Error Handling

Edge Cases: Index must fall within table bounds (0 to rows.length - 1). Deleting the last row of a table leaves it structurally empty, which might alert alerts in PowerPoint.

Errors: Throws RangeError if rowIndex is out of bounds.

Code Example

ppt.removeTableRow('SalesTable', 2); // Remove the third row (0-based)
