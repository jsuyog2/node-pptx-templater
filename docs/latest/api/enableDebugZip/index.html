enableDebugZip() API Method - node-pptx-templater

Description: Enables ZIP debug output. When enabled, every call to `toBuffer()` or `toStream()` will log all ZIP entries (name, compression method, sizes, CRC).

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Returns

PPTXTemplater — this (chainable)
