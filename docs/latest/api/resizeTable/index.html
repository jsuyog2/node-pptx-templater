resizeTable() API Method - node-pptx-templater

Description: Resizes a table to the specified width and height in EMUs. 1 inch = 914,400 EMUs.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
width number New width in EMUs.
height number New height in EMUs.

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Modifies underlying OpenXML nodes to reflect updates on slide serialization.

Edge Cases & Error Handling

Edge Cases: No critical edge cases documented. Verify argument boundaries.

Errors: Wrap inside try/catch blocks to process execution errors.

Code Example

ppt.resizeTable('SalesTable', 6858000, 1371600); // 7.5" wide × 1.5" tall
