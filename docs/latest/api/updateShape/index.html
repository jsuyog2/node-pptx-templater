updateShape() API Method - node-pptx-templater

Description: Updates an existing shape in-place.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
shapeId string
options Object

Returns

this — The chainable presentation templater instance.

OpenXML Impact

Updates shape geometry attributes, fills, borders, text body values, or shadows in slide XML in-place.

Edge Cases & Error Handling

Edge Cases: Dimensions/coordinates are converted where 1 pixel ≈ 9525 EMUs. Properties not specified are preserved.

Errors: Throws PPTXError if shape target is not found on targeted slide index.
