getDataLabels() API Method - node-pptx-templater

Description: Retrieves the current data labels configuration for a specific chart series.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
chartId string Chart name or shape ID.
[options] Object Options.
[options.series=0] number 0-based series index.

Returns

Promise<Object> — Current data label settings.

OpenXML Impact

Reads individual `<c:dLbl>` values and structures from `<c:dLbls>` inside slide chart XML caches.

Edge Cases & Error Handling

Edge Cases: Returns an empty array if the targeted series index has no custom data label overrides or does not exist.

Errors: Gracefully resolves any pending chart queue tasks first before executing to ensure read accuracy.

Code Example

const labels = await ppt.getDataLabels('revenue-chart', { series: 0 });
console.log(labels.showValue, labels.position);
