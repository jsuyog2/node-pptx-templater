getList() API Method - node-pptx-templater

Description: Retrieves list items from a shape or text box by name or placeholder tag.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tag string Shape name/ID or placeholder tag.

Returns

Array — Nested list structure of items.

OpenXML Impact

Reads individual paragraph structures, properties (a:pPr lvl), and combined run texts from shape text frames.

Edge Cases & Error Handling

Edge Cases: Scans specified shape or text placeholders, parsing nested levels and bullet definitions.

Errors: Returns empty array if shape has no text body or tag cannot be found.
