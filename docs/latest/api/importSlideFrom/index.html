importSlideFrom() API Method - node-pptx-templater

Description: Imports a single slide from another PPTXTemplater instance into this presentation. Preserves all slide layouts, charts, relationships, and embedded media.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
sourceEngine PPTXTemplater Source PPTXTemplater instance.
slideRef number|string Slide index (1-based), ID, or custom tag.

Returns

Promise<PPTXTemplater> — this (chainable)

OpenXML Impact

Remaps slide-level relationships to root presentation indices, duplicates layout links, and copies media files into the target ZIP package.

Edge Cases & Error Handling

Edge Cases: Source deck must be loaded first. Deduplicates layouts, media assets, and themes to prevent PowerPoint Repair Mode prompts.

Errors: Validates target slide index in the source presentation. Throws error if source presentation is invalid.
