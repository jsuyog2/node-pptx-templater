getCellPosition() API Method - node-pptx-templater

Description: Retrieves final rendered position of a table cell in pixels.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
rowIndex number 0-based row index.
colIndex number 0-based column index.

Returns

Object|null — Cell position { row, column, x, y } in pixels, or null.

OpenXML Impact

Reads cell positioning data in pixels. No write impact on slide XML.

Edge Cases & Error Handling

Edge Cases: Returns null if the slide table cannot be resolved.

Errors: Returns null if row or column indices exceed boundaries.
