replaceImage() API Method - node-pptx-templater

Description: Replaces an existing image in the presentation by shape name or relationship ID.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
imageIdOrName string Shape name, alt text, or relationship ID of the image.
sourcePathOrBuffer string|Buffer Path to the replacement image file, or a Buffer.

Returns

Promise<PPTXTemplater> — this (chainable)

OpenXML Impact

Overwrites target media target inside presentation archive or updates relationship mapping target references to point to the new image.

Edge Cases & Error Handling

Edge Cases: Image placeholder name/id must exist in template. Image types must match or content-type overrides must be registered.

Errors: Throws error if image reference is missing or if media buffer is corrupted.

Code Example

await ppt.replaceImage('company-logo', './new-logo.png');
