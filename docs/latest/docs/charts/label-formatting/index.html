Label Formatting - node-pptx-templater

Inline Custom Data Labels

In addition to using the updateDataLabels() method, node-pptx-templater supports defining custom data labels directly inline inside the standard updateChart() and updateChartData() series values arrays. This allows you to apply labels to chart series in a single step without introducing breaking API changes or calling separate methods.

Inline Object Syntax

To set custom labels inline, pass objects in the format { data: number, label: string } instead of standard numeric primitives inside the values array of a series:

  • data: The raw numeric value to write into the chart data cache and the embedded Excel worksheet (e.g. 145). Also accepts value as an alias (e.g. { value: 145, label: 'Custom' }).
  • label: The string label to display on the PowerPoint chart for that specific data point (e.g. '145 (30%)').

Validation Rules & Safety

To prevent PowerPoint Repair warnings and ensure data consistency, the engine automatically enforces the following validation checks:

  • Consistent Types: If one data point in a series defines a custom label, all other data points in that same series must also define a custom label (i.e. you cannot mix naked numbers and label objects in the same series values array).
  • Numeric Values: The data (or value) property of the label object must be a valid number.
  • String Labels: The label property must be a valid string.

Usage Example

const { PPTXTemplater } = require('node-pptx-templater');

async function main() {
  const ppt = await PPTXTemplater.load('template.pptx');
  
  // Update a chart and set custom labels inline
  ppt.useSlide(1).updateChart('RevenueChart', {
    categories: ['Q1', 'Q2', 'Q3', 'Q4'],
    series: [
      {
        name: 'Product A',
        values: [
          { data: 145, label: 'Q1: 145 (Low)' },
          { data: 210, label: 'Q2: 210 (Med)' },
          { data: 190, label: 'Q3: 190 (Med)' },
          { data: 250, label: 'Q4: 250 (High)' }
        ]
      }
    ]
  });
  
  await ppt.saveToFile('report.pptx');
}
main();
