Quick Start Guide - node-pptx-templater

Basic Placeholder Replacement Example

Below is a quick demonstration of loading a PowerPoint presentation, updating placeholders, and saving the output:

const { PPTXTemplater } = require('node-pptx-templater');

async function main() {
  const ppt = await PPTXTemplater.load('monthly_report_template.pptx');
  
  // Replace text placeholders on the first slide
  ppt.useSlide(1).replaceTextByTag('name', 'John Doe');
  
  // Save output presentation
  await ppt.saveToFile('output.pptx');
}

main().catch(console.error);
