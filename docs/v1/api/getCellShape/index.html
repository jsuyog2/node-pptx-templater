getCellShape() API Method - node-pptx-templater

Description: Discovers and retrieves details of an existing cell shape on the targeted slide.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
rowIndex number 0-based row index.
colIndex number 0-based column index.
shapeIndex number 0-based shape index in the cell.

Returns

Object|null — Shape details object, or null if not found.

OpenXML Impact

None (read-only query).

Edge Cases & Error Handling

Edge Cases: Retrieves details of shape in table cell.

Errors: Returns null if shape is not found.
