unmergeCells() API Method - node-pptx-templater

Description: Unmerges (splits) a previously merged cell region. Supports both positional arguments and an options object.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableIdOrOptions string|Object Table ID string, or options object.
[startRow] number 0-based start row of the merged region.
[startCol] number 0-based start column.
[endRow] number 0-based end row.
[endCol] number 0-based end column.

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Removes `gridSpan`, `rowSpan`, `hMerge`, and `vMerge` attributes from all target cells in the region.

Edge Cases & Error Handling

Edge Cases: Target must overlap an active merge region origin. Calling on an unmerged region acts as a no-op.

Errors: Gracefully ignores if cell coordinates do not match any active merge region.

Code Example

ppt.unmergeCells('SalesTable', 0, 0, 0, 2); // Unmerge region
// Via cell coordinate:
ppt.unmergeCells({ tableId: 'SalesTable', row: 0, col: 0 });
