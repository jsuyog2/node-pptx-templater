replaceMultiple() API Method - node-pptx-templater

Description: Replaces multiple text placeholder tags in a single pass. More efficient than calling `replaceTextByTag()` repeatedly.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
replacements Object<string, string> Map of `{ tag: value }` pairs.
[options] Object Options (same as `replaceTextByTag`).

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Modifies underlying OpenXML nodes to reflect updates on slide serialization.

Edge Cases & Error Handling

Edge Cases: No critical edge cases documented. Verify argument boundaries.

Errors: Wrap inside try/catch blocks to process execution errors.

Code Example

ppt.replaceMultiple({
'{{title}}': 'Q2 Report',
'{{date}}': 'June 2026',
'{{company}}': 'Acme Corp'
});
