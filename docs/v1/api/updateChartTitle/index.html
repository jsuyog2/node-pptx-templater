updateChartTitle() API Method - node-pptx-templater

Description: Updates only the title text of a chart.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
chartId string Chart name or shape ID.
title string New chart title.

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Finds and replaces text run values inside `<c:title>` and child `<a:t>` nodes under chart structure.

Edge Cases & Error Handling

Edge Cases: If the template chart has no initial title layout block, adding a title might require injecting new XML blocks.

Errors: Safely overrides title runs if title node exists; otherwise updates layout or prints warnings.

Code Example

ppt.updateChartTitle('revenue-chart', 'Q2 2026 Revenue');
