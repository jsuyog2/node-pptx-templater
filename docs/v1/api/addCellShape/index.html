addCellShape() API Method - node-pptx-templater

Description: Dynamically adds a shape inside a table cell based on cell coordinates.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
rowIndex number 0-based row index.
colIndex number 0-based column index.
options Object Shape configuration options.

Returns

this — The chainable presentation templater instance.

OpenXML Impact

Appends a `<p:sp>` shape node aligned to cell EMUs.

Edge Cases & Error Handling

Edge Cases: Index must fall within table bounds. Coordinates outside structural table dimensions align relative to cell borders.

Errors: Validates cell index before inserting.
