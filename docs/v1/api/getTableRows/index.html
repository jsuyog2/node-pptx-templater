getTableRows() API Method - node-pptx-templater

Description: Extracts table data from the active slide as structured JSON. The first row of the table is treated as the header row.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
[options] Object Extraction options.
[options.raw=false] boolean Return `string[][]` instead of object array.
[options.includeMetadata=false] boolean Return `{rows, rowCount, columnCount, mergedCells}`.

Returns

Array<Object>|Array<Array<string>>|Object — Extracted table data.

OpenXML Impact

Reads XML table data structure. No write impact on slide XML.

Edge Cases & Error Handling

Edge Cases: If tableId is not found, throws TableNotFoundError. Merged cells automatically resolve to their parent cell value.

Errors: Throws TableNotFoundError if table is not found, or PPTXError if slide selection is empty.

Code Example

// Object mode (default)
const rows = await ppt.getTableRows('SalesTable');
// → [{ region: 'North', sales: '1200' }, ...]
// Raw mode
const raw = await ppt.getTableRows('SalesTable', { raw: true });
// → [['North', '1200'], ...]
// Metadata mode
const meta = await ppt.getTableRows('SalesTable', { includeMetadata: true });
// → { rows: [...], rowCount: 5, columnCount: 3, mergedCells: [] }
