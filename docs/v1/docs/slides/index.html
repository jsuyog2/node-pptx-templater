Slides API Reference - node-pptx-templater

Duplicate, move, import, delete, and structure layout sections of slides.

Included API Methods

  • useSlide — Selects one or more slides to work on. All subsequent operations (replaceText, updateChart, etc.) apply to these slides. If not called, operations apply to ALL slides.
  • useAllSlides — Selects all slides.
  • addSlide — Adds a new slide to the presentation. Automatically generates required XML and relationship entries.
  • cloneSlide — Clones an existing slide and appends it to the end (or at a position).
  • removeSlide — Removes a slide from the presentation.
  • reorderSlides — Reorders slides in the presentation.
  • tagSlide — Tags a slide with a custom string identifier for later selection.
  • exportSlides — Exports selected slides to a new standalone PPTX engine. Useful for creating "slide decks" from a master template.
  • importSlideFrom — Imports a single slide from another PPTXTemplater instance into this presentation. Preserves all slide layouts, charts, relationships, and embedded media.
  • importSlides — Imports selected slides from the current template, discarding the rest. The remaining slides are reordered to match the provided array. Preserves all layouts, themes, relationships, and embedded media.
  • duplicateSlide — Delegates core actions to slide element sub-managers.
  • deleteSlide — Delegates core actions to slide element sub-managers.
  • moveSlide — Delegates core actions to slide element sub-managers.
  • insertSlide — Delegates core actions to slide element sub-managers.
  • getSlides — Delegates core actions to slide element sub-managers.
