Label Positions & Alignment

Adjust the placement of labels relative to data points or chart slices. Positioning options are fully PowerPoint-compatible and adapt to specific chart layouts (bar, line, pie, etc.).

Supported Position Options

  • center: Place labels directly centered on the data points.
  • insideEnd: Place labels inside the top/end border of the series shape.
  • insideBase: Place labels inside the base of the series shape.
  • outsideEnd: Place labels outside the top/end border of the series shape.
  • bestFit: Let PowerPoint automatically choose the best fit (recommended for Pie/Doughnut charts).
  • left / right / top / bottom: Position labels relative to the data point coordinates.

Positioning Example

ppt.useSlide(1).updateDataLabels('MarketSharePie', {
  series: 0,
  position: 'bestFit',
  showPercent: true
});
