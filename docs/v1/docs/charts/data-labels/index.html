Chart Data Labels - node-pptx-templater

Introduction to Chart Data Labels

Data labels make your PowerPoint charts easier to read by showing data values or custom annotations directly on the chart points. node-pptx-templater provides advanced data label support, including Excel-style Value From Cells, literal arrays, category mapping, dynamic templates, styling, and positioning.

Feature Highlights

  • Value From Cells: Reference cell ranges in the backing spreadsheet (e.g. Sheet1!D2:D10) as dynamic data labels.
  • Literal Arrays: Apply a list of static labels directly (e.g. ['High', 'Medium', 'Low']).
  • Category Mappings: Map specific category keys to custom label names (e.g. { 'Sales': 'Peak Performance' }).
  • Label Templates: Format labels dynamically using template placeholders like {category}, {value}, and {percentage}.
  • PowerPoint Stacking: Fully compliant with OpenXML layout and formatting rules to avoid PowerPoint Repair dialogs.

Value From Cells Code Example

Update a specific chart series to pull labels from a worksheet cell range:

const { PPTXTemplater } = require('node-pptx-templater');

async function main() {
  const ppt = await PPTXTemplater.load('template.pptx');
  
  // Set data labels for series 0 from cell range D2:D5
  ppt.useSlide(1).updateDataLabels('SalesChart', {
    series: 0,
    labelsFromCells: 'Sheet1!D2:D5'
  });
  
  await ppt.saveToFile('report.pptx');
}
main();
