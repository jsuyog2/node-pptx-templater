Text API Reference - node-pptx-templater

Execute text tag replacements, un-fragment text runs, search string tags, and apply links.

Included API Methods

  • replaceText — Replaces template placeholders (e.g., {{key}}) with values in the selected slides. Works inside text boxes, titles, grouped shapes, tables, and shapes.
  • addHyperlink — Adds or replaces a hyperlink on a text run or shape.
  • addSlideLink — Adds an inter-slide hyperlink to a specific text element.
  • addImageLink — Adds an inter-slide hyperlink to an image.
  • addShapeLink — Adds an inter-slide hyperlink to a shape.
  • addTextNavigationLink — Adds a special navigation link (next, previous, first, last slide) to a text element.
  • addShapeNavigationLink — Adds a special navigation link (next, previous, first, last slide) to a shape or image.
  • updateText — Updates shape text or list content by placeholder tag or shape name/ID. Supports bullet lists, numbered lists, nested lists, and custom styling.
  • getList — Retrieves list items from a shape or text box by name or placeholder tag.
  • validateList — Validates a list structure and values.
  • replaceTextByTag — Delegates core actions to slide element sub-managers.
  • replaceMultiple — Delegates core actions to slide element sub-managers.
  • findText — Delegates core actions to slide element sub-managers.
  • getTextElements — Delegates core actions to slide element sub-managers.
