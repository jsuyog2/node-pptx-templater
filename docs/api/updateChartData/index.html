updateChartData() API Method - node-pptx-templater

Description: Alias for `updateChart()`. Updates chart data for a named chart.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
chartId string Chart name or shape ID.
data Object Chart data object with `categories` and `series`.

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Rewrites numerical categories and series data caches in chart XML, and updates cell values in the backing `ppt/embeddings/*.xlsx` workbook in the ZIP archive.

Edge Cases & Error Handling

Edge Cases: Dataset categories and series coordinates must match initial chart mappings structure to avoid Excel workbook cell mismatches.

Errors: Throws ChartNotFoundError if chartId is missing. Verifies zip integrity before parsing.

Code Example

ppt.updateChartData('revenue-chart', {
categories: ['Q1', 'Q2', 'Q3'],
series: [{ name: 'Revenue', values: [100, 150, 200] }]
});
