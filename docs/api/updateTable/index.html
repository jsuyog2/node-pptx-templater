updateTable() API Method - node-pptx-templater

Description: Replaces table rows with new data in the selected slide(s). Preserves borders, merged cells, fonts, colors, and alignment from the template.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Parameters

Parameter Type Description
tableId string Table name or shape ID.
rows string[][] 2D array of cell values (row × col).

Returns

PPTXTemplater — this (chainable)

OpenXML Impact

Updates `<a:t>` element text inside individual table cells under `<a:tc>`, adjusting `<a:tcPr>` border styles and alignment rules.

Edge Cases & Error Handling

Edge Cases: If tableId is not found, throws TableNotFoundError. Coordinates outside structural table dimensions fail silently or align incorrectly depending on row dimensions.

Errors: Wrap in try/catch block to catch TableNotFoundError or SlideNotFoundError if active slide selection is empty.

Code Example

ppt.updateTable('employees-table', [
['Name', 'Role', 'Department'],
['John', 'Engineer', 'Platform'],
['Jane', 'Designer', 'Product']
]);
