getPerformanceMetrics() API Method - node-pptx-templater

Description: Returns performance metrics collected since `enablePerformanceProfile()` was called. Includes timing for template load, XML parse, chart update, image update, ZIP generation, total elapsed time, and memory usage. `imageUpdateMs`, `zipGenerationMs`, `totalMs`, `memoryUsedMB`.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Returns

Object — Metrics object with `templateLoadMs`, `parseMs`, `chartUpdateMs`,

OpenXML Impact

Modifies underlying OpenXML nodes to reflect updates on slide serialization.

Edge Cases & Error Handling

Edge Cases: No critical edge cases documented. Verify argument boundaries.

Errors: Wrap inside try/catch blocks to process execution errors.

Code Example

ppt.enablePerformanceProfile();
await ppt.updateChart('chart1', data);
const metrics = ppt.getPerformanceMetrics();
console.log(`Total: ${metrics.totalMs}ms, Memory: ${metrics.memoryUsedMB}MB`);
