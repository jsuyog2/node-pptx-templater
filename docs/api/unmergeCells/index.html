unmergeCells() API Method - node-pptx-templater

Description: Delegates core actions to slide element sub-managers.

Class: PPTXTemplater | File: src/core/PPTXTemplater.js

Returns

PPTXTemplater — The fluent engine instance.

OpenXML Impact

Removes `gridSpan`, `rowSpan`, `hMerge`, and `vMerge` attributes from all target cells in the region.

Edge Cases & Error Handling

Edge Cases: Target must overlap an active merge region origin. Calling on an unmerged region acts as a no-op.

Errors: Gracefully ignores if cell coordinates do not match any active merge region.
