# Security Policy

## Supported Versions

Only the latest major version of `node-pptx-templater` receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `node-pptx-templater`, please do **not** open a public issue. Instead, report it directly by email or via GitHub's private vulnerability reporting feature.

We will acknowledge your report within 48 hours and provide a detailed timeline for addressing the issue.
